# 🏥 About Project

Delight Multi Speciality Hospital is a modern Hospital Management System developed using **ASP.NET MVC**, **C#**, **SQL Server**, **Bootstrap**, **HTML**, **CSS**, and **JavaScript**.

The application helps hospitals efficiently manage patients, doctors, appointments, billing, pharmacy, laboratory, and administrative operations through an easy-to-use interface.
